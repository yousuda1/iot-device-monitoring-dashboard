# IoT Device Monitoring Dashboard

A simple IoT dashboard demo built using Python Flask.

## Features

- Temperature monitoring
- Humidity monitoring
- Battery status
- Device online/offline indicator
- Live updating dashboard
- Temperature trend chart

## Technology

- Python
- Flask
- Chart.js
- HTML
- CSS
- JavaScript

## Run Locally

pip install -r requirements.txt

python app.py

Then open:

http://127.0.0.1:5000

## Future Integration

The dashboard structure is ready for integration with:

- IoT devices
- REST APIs
